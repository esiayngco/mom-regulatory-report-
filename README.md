# Regulatory Survey Findings Interactive Report

An interactive D3.js storytelling report analyzing regulatory survey findings across multiple accreditation surveys. This project explores recurring compliance findings, compares DNV and DOH survey results, and provides an interactive interface for identifying trends and opportunities for quality improvement.

## Features

- Interactive D3.js visualizations
- DNV vs DOH comparison
- Repeat findings analysis
- Department-level exploration
- Searchable and filterable data table
- Responsive storytelling layout

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- D3.js v7

## Running the Project

Clone the repository and open the project using a local development server (such as VS Code Live Server).

```bash
git clone https://github.com/esiayngco/mom-regulatory-report-.git
```

Then launch `index.html` using Live Server.

## Dataset

The dashboard uses an internal regulatory survey crosswalk dataset containing findings from accreditation surveys, including DNV and DOH survey results.
